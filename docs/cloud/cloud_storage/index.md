# Cloud Storage Fundamentals

Welcome to the Cloud Storage course. In this module, we explore the different storage technologies provided by cloud platforms and how to choose the right one for your application.

## Course Overview

This course covers the essential concepts of cloud storage, including:

- **Storage Types**: Object, File, and Block storage.
- **Architecture**: How cloud storage scales and provides durability.
- **Security**: IAM policies and access control for storage resources.
- **Practical Application**: Hands-on lab using Minio for object storage.

## Objectives

By the end of this course, you will be able to:

1. Differentiate between Object, File, and Block storage.
2. Design storage solutions based on performance, cost, and persistence requirements.
3. Configure IAM policies to secure storage buckets.
4. Deploy and interact with a local S3-compatible object storage (Minio).
